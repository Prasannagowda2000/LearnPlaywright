# LearnPlaywright

A comprehensive learning repository for Playwright and JavaScript fundamentals, designed for beginners at The Testing Academy.

## 📚 Project Structure

This repository is organized into chapters, each covering specific topics:

### Chapter 01: Basics
Introduction to JavaScript and Playwright basics including:
- `01_Basics.js` - Getting started with JavaScript basics
- `02_JS.js` - JavaScript fundamentals
- `03_JS_Verify_Setup.js` - Verification of development setup
- `04_HotCode.js` - Hot code reloading examples

### Chapter 02: JavaScript Concepts
Deep dive into JavaScript concepts:
- `05_JS_Basics.js` - Core JavaScript basics
- `06_Identifier_Rules.js` - JavaScript identifier rules and conventions
- `07_Identifier_Part2.js` - Advanced identifier concepts
- `08_Comments.js` - JavaScript comments and documentation

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Prasannagowda2000/LearnPlaywright.git
```

2. Navigate to the project directory:
```bash
cd LearnPlaywright
```

3. Install dependencies (if needed):
```bash
npm install
```

## 📖 How to Use

Each JavaScript file in the chapters contains learning content and examples. You can run any file using Node.js:

```bash
node chapter_01_Basics/01_Basics.js
```

Follow the files in sequence within each chapter to build your understanding progressively.

## 🎯 Learning Path

Begin with **Chapter 01: Basics** to understand the fundamentals, then progress to **Chapter 02: JavaScript Concepts** for more advanced topics.
