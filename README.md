# LearnPlaywright

A comprehensive learning repository for Playwright and JavaScript fundamentals, designed for beginners at The Testing Academy.

## 📚 Project Structure

This repository is organized into chapters, each covering specific topics:

### Chapter 01: Basics
Introduction to JavaScript and Playwright basics including:
- `01_Basics.js` - Getting started with JavaScript basics
- `02_JS.js` - JavaScript fundamentals
- `03_JS_Verify_Setup.js` - Verification of development setup
- `04_HotCode.js` - Hot code reloading examples

### Chapter 02: JavaScript Concepts
Deep dive into JavaScript concepts:
- `05_JS_Basics.js` - Core JavaScript basics
- Variable declarations and scoping

### Chapter 03: Identifier & Literals
JavaScript identifiers and literals:
- `06_Identifier_Rules.js` - JavaScript identifier rules and conventions
- `07_Identifier_Part2.js` - Advanced identifier concepts
- `08_Comments.js` - JavaScript comments and documentation
- `js_identifier_rules.js` - Identifier rules reference

### Chapter 04: JavaScript Concepts (Advanced)
Advanced JavaScript concepts including:
- `09_var_let_const.js` - Variable declarations comparison
- `10_functions.js` - Function basics and scope
- `11_var_explained.js` - `var` keyword in depth
- `12_let_peope_love.js` - Why developers prefer `let`
- `13_const_explained.js` - `const` keyword explained
- `14_var_functionscope.js` - Function scope with `var`
- `15_let_scope.js` - Block scope with `let`
- `16_Hoisting.js` - JavaScript hoisting concept
- `17_hoisting_fn.js` - Function hoisting
- `18_let_hoisting.js` - `let` hoisting behavior
- `19_let_hoisting_block.js` - Block-level hoisting
- `20_let_const.js` - `let` vs `const` comparison
- `21_Jr_QA.js` - Junior QA reference materials

### Chapter 05: Literals
Understanding JavaScript literals:
- `23_null_undefined.js` - `null` and `undefined` differences
- `24_null.js` - `null` keyword detailed

### Chapter 06: Operators
JavaScript operators and operations:
- Arithmetic, logical, and comparison operators

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Prasannagowda2000/LearnPlaywright.git
```

2. Navigate to the project directory:
```bash
cd LearnPlaywright
```

3. Install dependencies (if needed):
```bash
npm install
```

## 📖 How to Use

Each JavaScript file in the chapters contains learning content and examples. You can run any file using Node.js:

```bash
node chapter_01_Basics/01_Basics.js
```

Follow the files in sequence within each chapter to build your understanding progressively.

## 🎯 Learning Path

Follow the chapters in order:
1. **Chapter 01: Basics** - JavaScript fundamentals and setup
2. **Chapter 02: JavaScript Concepts** - Core concepts and basics
3. **Chapter 03: Identifiers & Literals** - Naming conventions and literal values
4. **Chapter 04: JavaScript Concepts (Advanced)** - Variables, hoisting, scoping, and functions
5. **Chapter 05: Literals** - Understanding null and undefined
6. **Chapter 06: Operators** - Working with operators
